Скачать с сервера все обновления:
git status -s
если не было изменений то ок, иначе перетираем их:
git stash -u
далее:
git pull

Добавить все изменения на сервер
git add -A
Выполнить выполнить коммит. Описание обязательно.
git commit -m "commit reason text"
Отправить изменения на сервер
git push
Запустить миграции + сиды:
php artisan migrate:fresh --seed
Создать миграции + таблицы:
php artisan make:migration SectionsTables --create=sections

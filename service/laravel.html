Создать новый проект(загрузить с GitLab):
git clone https://gitlab.com/a.veneckiy/project.git .

Скачать с сервера все обновления:
git status -s
если не было изменений то ок, иначе перетираем их:
git stash -u
далее:
git pull

Добавить все изменения на сервер
git add -A
Выполнить выполнить коммит. Описание обязательно.
git commit -m "commit reason text"
Отправить изменения на сервер
git push

Запустить миграции + сиды:
php artisan migrate:fresh --seed
Создать миграции + таблицы:
php artisan make:migration SectionsTables --create=sections

Сгенерировать ключ:
php artisan key:generate

Запустить php под определенной версией:
/opt/php81/bin/php artisan migrate:fresh --seed

На фастпанели после загрузки фалов нужно сменить пользователя:
chown -R fastuser:fastuser .*

После установки нового проекта добавим вендоры:
php artisan vendor:publish
php artisan package:discover --ansi
composer require laravel/ui
php artisan ui bootstrap --auth
